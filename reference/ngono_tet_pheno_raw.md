# N. gonorrhoeae Tetracycline Resistance Phenotype Data Use Case 3

Tetracycline MIC data for 409 *Neisseria gonorrhoeae* isolates collected
in Eastern Spain between 2021 and 2024, used to investigate genetic
determinants of tetracycline resistance and implications for doxy-PEP
strategies. Described in Sánchez-Serrano *et al.* (2026)
<doi:10.1016/j.cmi.2025.12.026>.

## Usage

``` r
ngono_tet_pheno_raw
```

## Format

`ngono_tet_pheno_raw` A data frame with 409 rows and 2 columns:

- `id`: Sample identifier (ENA run accession).

- `Tetracycline`: MIC value in mg/L for tetracycline (numeric).
